安装HEXO

安装git

HEXO搭建远程网站需要git提供的免费仓库作为服务器，因此安装并配置git是有必要的。

  • 登录git官网下载Windos最新版本安装包。

  • 安装git软件，详情见Git 详细安装教程（详解 Git 安装过程的每一个步骤）_git安装-CSDN博客

  • 配置git

    • 配置用户名

    • 配置邮箱

    • 初始化本地仓库

    • 创建远程仓库

    • 配置ssh公钥

安装Node.js

Node.is是一个网络库，运行HEXO需要其支持因此需要安装。

  • 登录[Node.js官网](Node.js — 下载 Node.js® (nodejs.org)),选择Windos的预构建包方式安装。
  • 安装Node.js

安装HEXO

安装完git、Node.is后可通过Node.js提供的nmp命令安装HEXO。

  • 安装HEXO

$ npm install -g hexo-cli
